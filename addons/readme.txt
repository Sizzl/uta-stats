This folder contains released addons or community updates to UTStats.

These are not created by the UTStats team but by people who use it and
have released something that we think is worth including.

Included
========

EUT_Fix:
An updated actor by Matthew so you can decide which game types
to have UTStats log for (EUT provides its own UTStats compatible accuracy 
logger.

UTSTATS_Trigger:
A server mutator by +++crowbar+++ that will automatically updated your stats 
website.
